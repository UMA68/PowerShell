# ADR-9006: 非常時プロトコル（迷ったら戻る判断基準）

## Status

Accepted

## Context

運用が安定してくると、次のような「判断不能状態」が発生することがある。

- どこに書くべきか分からない
- Loop と GitHub の境界が揺らいだ気がする
- Copilot の提案を信じてよいか迷う
- 「統合しすぎたかもしれない」という違和感

この状態で判断を続けると、憲法違反や二重管理につながる。

## Decision

判断に迷った場合は、**作業を止めて以下に戻る**。

1. Single Source of Truth は GitHub か？
2. これは「思考中」か「再利用知」か「確定事項」か？
3. いま決めなくても破綻しないか？

上記に即答できない場合は：

- 何もしない
- OneNote に戻す
- 時間を置く

を正解とする。

## Consequences

### 👍

- 迷いが事故に発展しない
- 疲労時でも構造が壊れない

### 👎

- 進捗が一時的に止まる
