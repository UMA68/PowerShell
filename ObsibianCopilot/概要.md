# ObsidianのCopilot(Plugin)

## Index化時のコストをざっくり計算

Copilot Pluginで、OpenAIをAPI越しに利用している。

.mdファイル1つ当たりだいたい10KBだとして、
1ファイル当たり2600トークン位消費するだろうとてきとーに見積もった。

で、PowerShellの実行結果がこれ。

```PowerShell
Markdownファイルの総数: 1112
推定トークン数: 2891200
推定コスト:＄ 0.28912
```

実際Obsidianでインデックス作成を実施し、OpenAIのサイトで確認したところ、  
$0.3  
と表示された。
根拠は弱いけど、まぁまぁこれでヨシ！
